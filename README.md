# Lumière Skincare

A full-featured skincare e-commerce website built as a single-page experience — complete with product browsing, cart functionality, and a smooth checkout flow.

## Features

- 🛒 **Shopping Cart** — Add, remove, and update product quantities in real time
- 💳 **Checkout Modal** — Clean, integrated checkout experience without leaving the page
- 🧮 **GST & Shipping Calculations** — Automatic tax and shipping cost computation at checkout
- 🏷️ **Category Filtering** — Browse products by skincare category (cleansers, serums, moisturizers, etc.)
- 🖼️ **Product Image Carousels** — Multiple images per product with smooth carousel navigation
- 📱 **Responsive Design** — Works across desktop and mobile screens
